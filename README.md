# newcomer

GitHub 新手入门练习仓库，用于熟悉 Git 基本操作和 GitHub 核心功能。

## 练习内容

- **提交（Commit）**：学习如何撰写规范的提交信息
- **分支（Branch）**：创建分支、切换分支、合并分支
- **Pull Request**：发起 PR、Code Review 流程
- **Issue**：提交问题、使用标签管理
- **Fork & Clone**：从远程仓库 Fork 并克隆到本地
- **冲突解决**：处理 Merge Conflict 的基本方法

## 常用 Git 命令

```bash
# 克隆仓库
git clone https://github.com/laker1993/newcomer.git

# 查看状态
git status

# 提交变更
git add .
git commit -m "feat: add something"

# 推送到远程
git push origin main

# 拉取最新代码
git pull
```
